支持的功能

目前只支持串联，且不支持计算功能。

		本版本目标：
1、目前支持两种用电器：灯泡和电阻。
2、滑动变阻器可能会在将来版本或2.0之后的版本被支持。
3、目前仅支持两个用电器的串联或并联连接。
4、故障可以有0、1、2处，都在用电器上。
5、0处故障必须为动态电路，须设置开关的变化。
6、1处故障可以设置开关的变化也可以不设置，但请设置电表的变化。
7、2处故障可以有1-2处故障。
8、并联电路不允许设置短路用电器。
9、串联电路不允许同时设置短路。
10、计算前请给出题目中包含的全部条件：开关的变化、灯泡的亮暗、电表的变化、示数情况。
11、将来的版本将会支持的功能：滑动变阻器、复杂电路、电表、电源、开关的故障。
